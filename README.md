# 🌸 Iris Dataset Analysis Project

## 📌 Overview
This project performs exploratory data analysis (EDA) on the famous Iris dataset using Python in Google Colab. The aim is to understand patterns and relationships between different flower features and visualize how they contribute to species classification.

---

## 📊 Dataset Information
The Iris dataset contains 150 samples of iris flowers with the following features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species (Setosa, Versicolor, Virginica)

---

## 🛠️ Tools & Libraries Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn

---

## 📈 Project Steps

1. Load dataset
2. Data exploration and inspection
3. Data cleaning (if needed)
4. Statistical summary
5. Data visualization:
   - Scatter plots
   - Histograms
   - Pairplots
   - Correlation heatmap
6. Feature relationship analysis
7. Conclusion and insights

---

## 🔍 Key Insights

- Petal length and petal width are the most important features for distinguishing species.
- The Setosa species is clearly separable from the other two species.
- Versicolor and Virginica show some overlap in features.
- Petal features are more useful than sepal features for classification.
- Strong correlations exist between petal measurements.

---

## 🎯 Conclusion

This project demonstrates how exploratory data analysis and visualization help uncover patterns in data. The Iris dataset is a simple yet powerful dataset for understanding classification problems in machine learning.
